 # 🌗 Calculator : 

   A modern, neumorphic-style calculator with smooth day–night theme switching made with HTML, CSS, JavaScript.

   ------------------------------------------------------------------------------------------------------------------

 ## 📌 Overview : 

   This project features a visually appealing calculator UI inspired by neumorphism, combined with a dynamic day–night theme toggle.
   It is lightweight, responsive, and easy to use.

   ------------------------------------------------------------------------------------------------------------------
   🚀 **Live Demo** : [Calculator](https://basic-themed-calculator.vercel.app/) 

   ---

 ## ✨ Features :

   - Day & Night mode toggle
  
   - Neumorphic UI with soft shadows
  
   - Clean, modern layout
  
   - Standard arithmetic operations
  
   - AC, DEL, %, decimal, and double zero (00)
  
   - Smooth button interactions
  
   - Fully responsive design
  
   ------------------------------------------------------------------------------------------------------------------

 ## 🛠️ Tech Stack :

   - HTML5 – Structure
  
   - CSS3 – Neumorphism + Themes
  
   - JavaScript – Logic + Functionality
  
   - GitHub – Version control & hosting
  
   ------------------------------------------------------------------------------------------------------------------

 ## 🧮 Calculator Logic :

   - Handles continuous operations
  
   - Prevents illegal input sequences
  
   - Supports decimals and percentage
  
   - Auto-updates calculation display
  
   ------------------------------------------------------------------------------------------------------------------
